A list of all tiles and what they do:

Folder Listing
==============
Lists title of folders from a specified location. Useful to create subtopic
listing.

Daviz Grid
==========
Shows a grid with daviz preview + embed. Uses a Sparql query to generate this.

Visualizations Row
==================
Shows a single row with previews of Daviz (from Sparql), Graphs and Indicators
(from ElasticSearch).

Daviz Embed
===========
Embed a single Daviz (by specifing a URL)

Daviz Preview
=============
Preview a single Daviz (by specifing a URL)

Catalogue Teaser
================
Teaser for results in Bise Catalogue
